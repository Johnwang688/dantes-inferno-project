# A Taste of Judgement: Dante's Inferno as Dessert

An interactive web project for Honors English II that represents Dante's *Inferno* as a layered dessert, where each layer symbolizes a different circle of Hell.

## Project Overview

This project explores Dante's depiction of Hell as a descending structure of worsening sin and punishment. The dessert uses increasingly unpleasant textures and temperatures—progressing from mild and tasteless to deceptively sweet to burning heat and finally frozen isolation—to mirror how sins become more deliberate, evil, and further from God's love as one descends.

## The Layers

| Circle | Dessert Layer | Symbolism |
|--------|---------------|-----------|
| **Limbo** | Plain whipped cream | Light and flavorless, like suspended longing |
| **Gluttony** | Soggy bread pudding | Excessive indulgence in earthly pleasures |
| **Greed** | Dense nut-filled brownie | Heavy, hoarded wealth |
| **Violence** | Spicy chili chocolate ganache | Burning punishment for harmful acts |
| **Fraud** | Vanilla custard with hidden bitter chocolate | Deceptive sweetness concealing bitter truth |
| **Treachery** | Hard-frozen vanilla ice cream | Frozen isolation, furthest from divine warmth |
